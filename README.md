# Travle-journal Website

## Project Overview
The travel journal website is respectivels used to identify the verious place one have been each year in different seasons

## Features
- Add new travel entries

- Capture location and landmark details

- Record travel date

- Specify time of the year (Spring, Summer, Autumn, Winter)

- Write personal travel notes

- Display travel entries dynamically

- List visited places

## Technologies Used
travel-journal/
│── index.html
│── style.css
│── script.js
│── README.md


## How to Use
1. The form collects user input.

2. JavaScript listens for the submit event.

3. When submitted:

         The default page refresh is prevented.

         A new list item is created.

         The travel entry is appended to the page dynamically.
